The span and div elements are both generic container elements in HTML. However, they have different display properties.

The span element is an inline element, which means that it flows with the surrounding text. The div element is a block-level element, which means that it starts on a new line and takes up the full width of its container.

Here is a table that summarizes the key differences between span and div elements:

Property span div
Display Inline Block
Newline No Yes
Width Content-width Container-width

When to use span and div elements:

  • Use the span element to group inline elements, such as text or links.
  • Use the div element to group block-level elements, such as headings, paragraphs, or images.
